\section{Methodology}
\label{sec:methodology}
\para{Overview.}
We run deterministic, inference-only API probes with temperature $=0$ to measure functional representation quality without model retraining. We evaluate two models (\gptfourone, \gptfouronemini), two prompt conditions (direct, reasoned), and four tasks: \digitword, \worddigit, \patterncls, and \mgsmfr.

\para{Datasets.}
Our core dataset is a synthetic French numeral set covering all integers from 0 to 199, generated by canonical rules in \texttt{src/french\_numerals.py}. This yields 200 unique examples with no duplicates. We partition by number band: 0--69, 70--99, and 100--199. For broader reasoning, we evaluate a French MGSM subset (\texttt{datasets/mgsm/mgsm\_fr.tsv}) with 250 rows; missing question rate is 0\%, missing answer rate is 1.6\%, and no duplicate questions were detected.

\para{Scoring and normalization.}
For \digitword, we score exact and normalized matches after lowercasing and hyphen/spacing normalization. For \worddigit, we parse predicted integers using regex and score exact integer match. \patterncls is scored by exact class label match. We treat 70--99 as the irregular band and aggregate 0--69 plus 100--199 as regular for significance tests.

\para{Statistical analysis.}
We compute accuracy by task, band, model, and condition. For each complete run, we test irregular-vs-regular differences with two-proportion z-tests, report risk differences and confidence intervals, and apply Benjamini--Hochberg FDR correction at $q=0.05$. We also fit a pooled logistic model:
\[
\texttt{correct} \sim \texttt{irregular} + C(\texttt{model}) + C(\texttt{condition}) + C(\texttt{task}).
\]

\para{Implementation details.}
We use Python 3.12.8 with \texttt{openai} 2.26.0, \texttt{pandas} 3.0.1, \texttt{numpy} 2.4.2, \texttt{scipy} 1.17.1, \texttt{statsmodels} 0.14.6, \texttt{matplotlib} 3.10.8, and \texttt{seaborn} 0.13.2. Generation uses \texttt{temperature=0}, \texttt{max\_output\_tokens=180}, and \texttt{seed=42}. Raw outputs are logged in \texttt{results/raw/raw\_outputs\_20260308\_214204.jsonl}.

\para{Baselines.}
Our principal baseline is the direct prompt condition; the reasoned condition is an ablation on output style. We do not include training-based baselines because the study targets black-box API behavior under controlled prompting.
